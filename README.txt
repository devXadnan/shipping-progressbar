# shipping-progressbar
Shipping progress bar at cart. I can be used in product page, cart page and cart drawer (highly recommended). 

********************************

How to Install it

*******************************

1- Add Code in cart-drawer.liquid
Search for cart-drawer-items and add code from "cart-drawer.liquid" file.

2- Create a new snippet with the name shipping-progress-bar.liquid
Add the code from "shipping-progress-bar.liquid" to that snippet.

3-: Add schema code
Add the code from schema.json in config/settings_schema.json file
